02 — Proyectos
Proyecto 01

NexusBank — Aplicación Web de Banca Virtual

GitHub

Aplicación bancaria full stack con autenticación JWT, gestión de cuentas, movimientos financieros categorizados y simulador de escenarios financieros personales.

Funcionalidades

  • Sistema de autenticación y autorización con JWT (registro, login y rutas protegidas)
  • Gestión de cuentas bancarias con validación de unicidad y control de saldos en tiempo real
  • Módulo de movimientos financieros: transferencias, ingresos y egresos con 13 categorías personalizadas
  • Modelado de datos relacional con migraciones versionadas y transacciones atómicas
  • Simulador financiero con proyección de ahorro, cálculo de préstamos e inversión con interés compuesto
  • Historial paginado con filtros por tipo de movimiento y categorización de gastos

Impacto

  • Solución funcional end-to-end que cubre el ciclo completo de gestión financiera personal
  • Arquitectura en capas (Controllers → Services → Data) escalable y mantenible
  • Mayor trazabilidad financiera con categorización de ingresos y egresos
  • API REST con 8 endpoints documentados y manejo centralizado de errores

Proyecto 02

News Explorer — Búsqueda y Guardado de Noticias

Demo en vivo GitHub Frontend GitHub Backend

Aplicación web full stack para búsqueda de noticias en tiempo real y gestión personalizada de artículos guardados, con autenticación segura y API REST propia.

Funcionalidades

  • Sistema de autenticación y autorización con JWT (registro, login y rutas protegidas)
  • Búsqueda de noticias en tiempo real integrada con NewsAPI
  • Guardado y eliminación de artículos por usuario con control de propiedad
  • Página de artículos guardados con palabras clave ordenadas por popularidad
  • Validación instantánea de formularios con hook personalizado
  • API REST propia con 6 endpoints para gestión de usuarios y artículos

Impacto

  • Solución funcional end-to-end que cubre el ciclo completo de búsqueda y gestión de noticias
  • Arquitectura modular (Controllers → Routes → Models → Middlewares) escalable
  • Experiencia de usuario mejorada con validación en tiempo real y persistencia de búsquedas
React Node.js Express MongoDB JWT BCrypt API REST Vite Vercel Railway
